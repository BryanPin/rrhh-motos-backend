Sistema completo de gestión de recursos humanos para microempresa de repuestos de motos con Node.js, Express y PostgreSQL.
 Características:

 Gestión de Empleados: CRUD completo, perfiles, historiales
 Control de Asistencia: Check-in/check-out, cálculo de horas, horas extras
 Solicitudes y Permisos: Vacaciones, permisos médicos, calamidades
 Nómina: Cálculo automático con comisiones, deducciones, IESS
 Ventas: Registro de ventas con cálculo de comisiones
 Dashboard: Estadísticas en tiempo real
 Autenticación JWT: Sistema seguro de login
 Roles de Usuario: Admin, Supervisor, Empleado

 Tecnologías Utilizadas:

Node.js - Runtime de JavaScript
Express - Framework web
PostgreSQL - Base de datos relacional
pg - Cliente PostgreSQL para Node.js
bcryptjs - Hash de contraseñas
jsonwebtoken - Autenticación JWT
express-validator - Validación de datos
dotenv - Variables de entorno
cors - Control de acceso CORS
morgan - Logger HTTP
